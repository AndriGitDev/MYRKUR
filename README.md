Hey there! MYRKUR (Icelandic for "darkness," fitting, right?) is a personal project I'm building to visualize simulated cyberattacks targeting Iceland. The idea is to create a dynamic and engaging map that shows where (simulated) attacks are coming from, what types of attacks they are, and get a general overview of the (again, simulated) cyber threat landscape.

It's all for fun and learning, using a mix of JavaScript for the simulation and Leaflet.js to bring the map to life. All the data you see is **simulated** based on internet averages and general knowledge â€“ no real attacks here!
